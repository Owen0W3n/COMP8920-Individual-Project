Keyboard Store Python Application

Files:
- main.py
- store.py
- inventory.py
- cart.py
- products.py

How to run:
1. Put all .py files in the same folder.
2. Open terminal in that folder.
3. Run:
   python main.py

Notes:
- inventory.json will be created automatically on first run.
- Inventory is saved after adding products and after successful checkout.
